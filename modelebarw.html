Modele barw

Rodzaje modeli barw w CSS

Kolory w CSS można określić nastepującymi modelami:

  • HEX (Kod szesnatkowy)
  • HEX z przezroczystością
  • RGB (Red, green, blue)
  • RGBA (Red, green, blue, alpha)
  • HSL (Hue, saturation, lightness)
  • HSLA (Hue, saturation, lightness, alpha)
Kliknij na dany model aby przenieść się do jego opisu...

Kliknij tutaj aby przejść do generatora losowo pokolorowanych kwadratów

HEX

Kolor w kodzie szesnatkowy jest określany za pomocą
#RRGGBB, gdzie liczby całkowite szesnatkowe RR (red) GG (green) i BB (blue) określają składowe koloru.
Wszystkie wartości muszą zawierać się w przedziale od 00 do FF

Np. #00ff00 da nam kolor zielony gdyż wartość koloru zielonego jest najwyższą możliwą
a reszta wartości jest równa zeru. Poniżej znajduje się kilka innych przykładów.

h1 { color: #ff0000 } /* znacznik h1 otrzyma czerwony kolor tekstu */
h2 { color: #0000ff } /* znacznik h2 otrzyma niebieski kolor tekstu */
h3 { color: #ffffff } /* znacznik h3 otrzyma biały kolor tekstu */

HEX z przezroczystością

Aby określić kolor w kodzie szesnatkowym z dodatkiem przezroczystości, do poznanego przed chwilą formatu
#RRGGBB dodajemy dwie kolejne cyfry których wartość musi zawierać się w przedziale od 00 do FF

p { color: #ff000080 } /* znacznik p otrzyma przezroczysty czerwony kolor tekstu */

RGB

Wartość koloru RGB określa się za pomocą funkcji rgb(), której składnia jest następująca:
rgb(red,green,blue)

Każdy parametr (czerwony, zielony i niebieski) określa intensywność koloru i może być liczbą całkowitą
z zakresu od 0 do 255 lub wartością procentową (od 0% do 100%).
Na przykład wartość rgb(0,0,255) jest renderowana jako kolor niebieski, ponieważ parametr niebieski
jest ustawiony na najwyższą wartość (255), a pozostałe wartości są ustawione na 0.

Fakt że parametry mogą być liczbami całkowitymi z zakresu od 0 do 255 i wartościami procentowymi
oznacza że zapis rgb(0,0,255) i rgb(0%,0%,100%) da ten sam rezultat.

.box { color: rgba(0,255,0) } /* elementy z klasą box otrzymają zielony kolor tekstu */

RGBA

RGBA to model RGB z dodatkiem przezroczystości. Aby dodać przezroczystość, zmieniamy składnię
rgb(red,green,blue) na rgba(red,green,blue,alpha).
Alpha (kanał alfa) oznacza przezroczystość i wartość tego parametru może mieć wartość od 0 do 1

span { background-color: rgba(0,255,0,0.5) } /* znacznik span otrzyma półprzezroczyste zielone tło */

Poniżej znajduje się przedstawienie modelu kolorów RGBA.
Skrypt w języku Javascript generuje bloki w wybranym kolorze z wartościami przezroczystości od 0.1 do 1
Najedź na kwadrat aby zobaczyć jego wartość przezroczystości

HSL

HSL to skrót od słów hue (barwa), saturation (nasycenie) i lightness (jasność), który oznacza
cylindryczną reprezentację kolorów we współrzędnych.

Wartość koloru HSL określa się za pomocą funkcji hsl(), która ma następującą składnię:
hsl(hue, saturation, lightness)

Barwa to stopień na kole kolorów (od 0 do 360) - 0 (lub 360) to kolor czerwony, 120 to kolor zielony, 240 to kolor niebieski.
Nasycenie to wartość procentowa; 0% oznacza odcień szarości, a 100% to pełny kolor.
Jasność to również wartość procentowa; 0% to czerń, a 100% to biel.

.container { background-color: hsl(120,70%,50%) } /* elementy z klasą box otrzymają lekko szarawo zielony kolor tła */

HSLA

Wartości kolorów HSLA są rozszerzeniem wartości kolorów HSL o kanał alfa, który określa nieprzezroczystość obiektu.

Wartość koloru HSLA określa się za pomocą funkcji hsla() o następującej składni: hsla(hue, saturation, lightness, alpha)

Parametr alfa jest liczbą z przedziału od 0,0 (całkowicie przezroczysta) do 1,0 (całkowicie nieprzezroczysta).

section { background-color: hsla(0,0%,0%,0.5) } /* znacznik section otrzyma pół przezroczyste czarne tło */
